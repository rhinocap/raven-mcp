## This session

### Claude Desktop Extension (MCPB) packaging
**What:** Packaged Raven as an `.mcpb` (MCP Bundle, formerly DXT) so Claude Desktop users can install in one click — no JSON editing, no Node install, no terminal.
- Added `manifest.json` at repo root (manifest_version 0.3) listing all 15 tools, icon, compatibility matrix.
- Added `scripts/build-mcpb.sh` — clean build pipeline: TS compile → stage dist/ + src/data/ + minimal package.json → install production-only deps → `npx @anthropic-ai/mcpb pack`.
- Added `npm run build:mcpb` script.
- Output: `site/raven.mcpb` (4.2 MB) served at `https://ravenmcp.ai/raven.mcpb`.
- Gitignored `.mcpb-stage/` to keep the staging dir out of commits.
- Verified the bundled server boots and lists all 15 tools via MCP initialize + tools/list stdio roundtrip.

**Why:** Users were struggling to install Raven on Claude Desktop. Editing `claude_desktop_config.json`, dealing with Node/npx resolution, and restarting the app was breaking people. The `.mcpb` format is Anthropic's official single-file extension — drag/drop install. The npm/npx install path is unchanged for Claude Code users.

**Pushed:** pending commit
